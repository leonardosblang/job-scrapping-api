# API e testes unitários da API Job Scrapping

## Instação e rodar

Instalar dependências

```bash
pip install fastapi
pip install uvicorn[standard]
pip install pymongo
```

Rodar API

```bash
uvicorn main:app
```

Rodar Testes

```bash
python -m unittest
```
